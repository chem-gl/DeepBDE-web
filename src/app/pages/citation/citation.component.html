📋 Usage Guidelines

🎯 When to Cite

  • When using DeepBDE predictions in your research
  • When comparing with other BDE prediction methods
  • When building upon our methodology
  • When using our molecular editor or API

📝 Citation Best Practices

  • Include the full citation in your references
  • Mention DeepBDE in your methodology section
  • Specify which version you used
  • Include relevant parameters or settings

🔗 Linking Guidelines

  • Link to our GitHub repository for code access
  • Reference the web application for interactive use
  • Include DOI when available
  • Mention funding sources when applicable
